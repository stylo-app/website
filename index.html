Stylo

Use a smartphone as a hardware wallet to manage Polkadot, Kusama, and other crypto accounts.

Download from GitHub
Download from Play store

Stylo is a mobile application meant to be installed on a phone that should remain offline. Stylo allows to create, recover and manage accounts. Using QR codes to scan a transaction, the app allows to review and sign any transaction offline.

Learn how to use it by following a tutorial: Using Stylo for Polkadot or Kusama.

This application is open source and the code is available on GitHub.
